# 💤 NeoVim Configuration

My neovim config, extended from [LazyVim](https://github.com/LazyVim/LazyVim).

## Install

```bash
# Backup your stuff
mv ~/.config/nvim{,.bak}
mv ~/.local/share/nvim{,.bak}
mv ~/.local/state/nvim{,.bak}
mv ~/.cache/nvim{,.bak}

git clone https://github.com/Raphael-Soares/nvim.git ~/.config/nvim
```
